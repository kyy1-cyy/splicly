üöÄ Getting Started

How to install and Run your VR games

‚ñ∂

Step 1: Join our Discord

Head to our Discord, this is the main thing you'll need:

Join our Discord Server

‚ñ∂

Step 2: Find Your Game

Head over to the #free-vr!2 channel. We do have a #free-vr channel, but the !2 version is much easier to use.

When in that channel, open the link to access our games list. Click on a game to see the download link.

If you don't find your game, go to #request-games, click the link, and request your game by providing the title and meta game link (optional).

‚ñ∂

Step 3: Download Your Game

Once your game has been added if requested, go back to the !2 channel:

  1. Click on the game you want
  2. Click the Google Drive download link
  3. You'll find a .apk file and a folder containing the game's .obb files
‚ñ∂

Step 4: Install the Game

After downloading the files/folder, follow these steps to install the game:

  1. Install the .apk file
  2. Place the .obb folder in the correct location (Android/obb/pasteHere)

Important: Make sure it's the whole folder that contains the .obb files, don't paste the .obb files directly.

Note for Quest 2 users: If you're on Quest 2 (not Quest 3), you'll need to watch this video to learn how to properly paste the game's .obb files.

‚ñ∂

Step 5: Launch and Play

To launch your game:

  1. Open your game library
  2. Look for the folder with a question mark icon on the left
  3. Find and click on the game you just installed

That's it! Your game should now be installed and ready to play.

Need Additional Help?

If you encounter any issues, please visit our Discord server and ask for assistance in the appropriate help channel.
